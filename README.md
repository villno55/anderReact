Taller React JS

Proyecto desarrollado en React JS con Vite.

Este proyecto contiene 10 ejercicios prácticos donde se aplican los conceptos fundamentales de React:

- Variables en JSX
- Funciones
- Props
- Desestructuración
- Arreglos
- Método map()
- Renderizado condicional
- Operador ternario
- Componentes padre e hijo

El proyecto puede ejecutarse localmente con:

npm install
npm run dev

O visualizarse directamente en el navegador:

https://villno55.github.io/anderReact/
